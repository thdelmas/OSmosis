# Game Consoles & Media Devices

## Overview

Game consoles and streaming devices range from fully open (Steam Deck) to heavily locked (modern PlayStation/Xbox). OSmosis documents the flashable/moddable subset and lists locked devices for awareness.

## Categories

### Steam Deck
Valve's Steam Deck runs SteamOS (Arch Linux). Users can install any Linux distro, dual-boot Windows, or customize SteamOS. Standard SD card / USB boot.

**Status:** Planned

### Nintendo Switch
Unpatched V1 Switch units (Erista, pre-mid-2018) can run custom firmware (Atmosphere) via the RCM exploit. Patched units and Switch Lite are locked.

**Status:** Research (Erista only)

### Retro Consoles & Handhelds
See [Retro Handhelds & Arcade](retro-handhelds.md) for Anbernic, Miyoo, MiSTer FPGA, etc.

### Streaming Devices
- **Chromecast with Google TV** — runs Android TV, can sideload apps via ADB
- **Amazon Fire TV Stick** — runs Fire OS (Android fork), supports ADB sideload
- **Roku** — proprietary, not supported
- **Apple TV** — proprietary, not supported

**Status:** Planned (Android-based); Not supported (Roku, Apple TV)

### E-Readers
See [E-Readers](e-readers.md) for Kobo, Kindle, reMarkable.
